The font for IT Professionals!

icons for all your needs

  1. Download or Install

    Via GitHub Via NPM
    npm install --save bc-font

  2. Link the CSS

    Within a module: import 'bc-font/dist/bc-font.css';
    Or within your HTML: <link rel="stylesheet" href="node_modules/bc-font/dist/bc-font.css" />

  3. Add your first icon

    <i class="icon icon-java"></i>
